#模板结构

CrossPHP默认的模板语言为PHP本身,一个app模板结构如下:

	├─templates 模板文件夹
	│  ├─default
	│  │   ├─acl
	│  │   │   ├─list.tpl.php
	│  │   │   ├─index.tpl.php  
	│  │   │   └─add.tpl.php
	│  │   │    
	│  │   ├─main
	│  │   ├─security
	│  │   ├─...
	│  │   └─default.layer.php 默认布局文件
	│  └─web
	│      ├─acl
	│      ├─main
	│      ├─security
	│      ├─...
	│      └─default.layer.php 默认布局文件


1. 模板命名方式为目录和控制器中的类一一对应, 模板文件和控制器中的方法对应,即模板名称/控制器名称/方法名称.
2. 布局文件放在模板文件的根目录下.
